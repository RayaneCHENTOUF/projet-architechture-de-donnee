# Assets pour la présentation

Ce dossier contient les schémas recommandés pour ton PowerPoint.

## Schémas disponibles

1. `architecture_overview.mmd`
   - Schéma global du projet
   - Slide recommandée : architecture globale
   - Montre : sources, Bronze, Silver, Gold, MySQL, MongoDB, API, Front

2. `etl_pipeline.mmd`
   - Schéma du pipeline de transformation
   - Slide recommandée : pipeline ETL
   - Montre : scripts Silver et Gold et flux de fichiers

3. `storage_architecture.mmd`
   - Schéma du stockage relationnel / NoSQL
   - Slide recommandée : stockage / modélisation
   - Montre : MySQL pour KPI, MongoDB pour GeoJSON

4. `api_architecture.mmd`
   - Schéma de l’API et de la consommation front
   - Slide recommandée : API et interopérabilité
   - Montre : FastAPI, MySQL, MongoDB, endpoints, front

## Notes

- Aucune image existante n’a été trouvée dans le dépôt (`*.png`, `*.jpg`, `*.svg`).
- Ces fichiers sont des sources Mermaid : tu peux les transformer en images via un outil compatible (Mermaid Live Editor, draw.io, Obsidian, ou un convertisseur Mermaid CLI).
- Si tu veux, je peux aussi générer un fichier `presentation_assets/slide_plan.md` avec le plan de contenu pour chaque slide.
